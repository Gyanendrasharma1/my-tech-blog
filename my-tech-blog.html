Programming Languages

  • HTML

    HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the basic structure of a website, which is enhanced and modified using CSS and JavaScript. HTML uses "tags" to define elements on a webpage.

Theory for the Given Basic HTML Code

<!DOCTYPE html>

  • Definition:Declares the document type and version of HTML.
  • Purpose:Ensures the browser interprets the document as an HTML5 file.
  • Importance:It helps maintain consistency and compatibility across web browsers.

<head>

  • Definition:Contains metadata about the document, which is not visible on the webpage but is crucial for functionality.
  • Elements in<head>
    • <meta charset="UTF-8">
      • Specifies the character encoding (UTF-8 supports most characters and symbols).
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">:
      • Ensures the webpage scales correctly on different devices, especially mobile phones.
      • The attribute width=device-width sets the width of the page to the device's screen width.
      • initial-scale=1.0 ensures the initial zoom level is 100%.
    • <title>Document</title>
      • Sets the title of the webpage, displayed on the browser tab.
      • It is important for SEO and user navigation.

<body>

  • Definition:Contains all the visible content of the webpage.
  • Role:Any text, images, videos, or interactive elements that appear on the webpage are placed within the <body> tags.

Conclusion

  • This structure is a starting template for any HTML document. Developers add content and elements within the <body> tag to build functional webpages. The <head> tag ensures the page is well-optimized and ready for different devices and browsers.
